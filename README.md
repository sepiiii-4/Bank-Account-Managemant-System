# 🏦 Bank Account Management System (Version 2)

A desktop banking management application developed with **Python**, **Tkinter**, and **SQLite**.

This project is the second and completely redesigned version of my previous Bank Account Management System. The main goal of this update was to improve the project's architecture by applying Object-Oriented Programming principles, implementing a **Layered Architecture**, and building a **Multi-Frame user interface** for better code organization, maintainability, and scalability.

---

## ✨ What's New in Version 2

Compared to the first version, this project includes:

- ✅ Layered Architecture
    - Presentation Layer
    - Business Logic Layer
    - Data Access Layer
    - Common

- ✅ Multi-Frame Tkinter UI

- ✅ Better Object-Oriented Design

- ✅ Improved input validation

- ✅ User authentication (Sign Up / Login)

- ✅ Deposit funds

- ✅ Withdraw funds

- ✅ Transfer funds between accounts

- ✅ Update account information

- ✅ Delete account

- ✅ Activate / Deactivate accounts

- ✅ Role management

- ✅ Improved Treeview management

- ✅ Cleaner, modular, and more maintainable code
